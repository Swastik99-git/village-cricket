## 📖 About

Village Cricket Management System is a platform designed to digitize local village cricket. It allows players and administrators to manage matches, player profiles, announcements, and statistics from a clean and easy-to-use interface.

The application is optimized for mobile devices so players and organizers can use it directly from their phones.

# 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/Swastik99-git/village-cricket.git
```

Go to the project directory

```bash
cd village-cricket
```

Install dependencies

```bash
npm install
```

Create a `.env` file

```env
VITE_SUPABASE_URL=YOUR_SUPABASE_URL
VITE_SUPABASE_ANON_KEY=YOUR_SUPABASE_ANON_KEY
```

Run the project

```bash
npm run dev
```
